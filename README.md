컴퓨터 네가지 핵심 부품
1. CPU - 메모리에 저장된 명령어를 읽어들이고 해석하고 저장하는 부품이다 
2. 메모리 - 현재 실행되는 프로그램의 명령어와 데이터를 정장하는 부분이다
3. 보조기억장치 - 전원이 꺼저도 보관할 프로그램을 저장하는 부품이다
4. 입출력장치 - 컴퓨터와 외부에 연결되어 컴퓨터 내부와 정보를 죠환할수 있는 부품이다
